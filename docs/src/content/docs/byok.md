---
title: Configure Providers (BYOK)
description: Configure BYOK and local providers in Refact.
---

Refact is BYOK/local-only. Add at least one provider before using chat, agents, or code completion.

## Add A Provider

1. Open **Provider Setup** or **Configure Providers**.
2. Choose a provider such as OpenAI, Anthropic, Gemini, DeepSeek, OpenRouter, Groq, xAI, Ollama, LM Studio, vLLM, or a custom OpenAI-compatible endpoint.
3. Enter the provider credentials or local endpoint details.
4. Enable the models you want Refact to use.

## Choose Defaults

Open **Default Models** and select defaults for:

- Chat.
- Light chat.
- Thinking/reasoning.
- Buddy.
- Code completion.
- Embeddings, if your workflow uses semantic search.

## Billing And Privacy

Refact does not charge for model usage. Requests are billed or limited only by the provider or local runtime you configure.

Requests are sent only to configured providers and integrations.
